# Bug Reference Photos

This folder contains screenshots and photos provided by Ali to illustrate bugs visually.

## Naming convention (after renaming)
`issue-NNN_description_NNN.jpg`

Example: `issue-001_panel-drifts-down_01.jpg`

## How to add photos
1. Ali drops the photo into this folder with any name
2. Claude renames it to match the convention above
3. Claude references it in the relevant BUGS.md entry
